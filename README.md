# Burst balloons

### Команды:

`npm start` - чтобы запустить проект

Программа работает по ссылке - localhost:8080

### Описание: 

Игра, в которой игроку необходимо лопнуть все шарики! 

### Что использовалось: 

webpack, нативный js, canvas
